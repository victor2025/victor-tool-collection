# 🌪 台风观测工具

中央气象台实时台风路径观测工具，通过 iframe 嵌入方式访问。

## 功能

- **实时台风路径** — 展示当前活跃台风的实时位置、强度和路径预测
- **中央气象台数据** — 数据来源：中国气象局中央气象台
- **多图层切换** — 支持卫星云图、雷达图、风场等多种图层叠加

## 技术栈

- 纯 HTML + CSS + JavaScript
- iframe 嵌入外部页面（https://typhoon.weather.com.cn）
- 无外部依赖
- 无构建步骤

## 目录结构

```
typhoon/
├── README.md

```

## 注意事项

- 目标站点可能设置了 `X-Frame-Options` 或 CSP 头，限制 iframe 嵌入
- 若嵌入失败，页面会自动显示「在新窗口打开」的链接
- 在 Tab 模式下，页面会自动隐藏「回到首页」链接
