# 🔐 管理后台 · 访问统计

密码保护的管理后台，用于查看 Victor Tool Collection 的访问统计数据和图表。

## 功能

- **密码登录** — 默认密码 `admin888`（登录后可在页面内修改）
- **概览卡片** — 总访问量、近 7 天、独立 IP、日均访问
- **各工具排行** — 横向柱状图展示每个工具的访问次数
- **时段分布** — 24 小时折线图，观察访问高峰时段
- **IP 明细** — 每个工具下各 IP 的访问次数
- **分页访问记录** — 支持工具筛选、页码跳转
- **设备标识** — 记录并展示浏览器设备唯一标识（localStorage UUID）
- **设备打标** — 点击设备标识可添加/修改/删除标签
- **标签模式切换** — 在显示 device_id 和显示标签之间切换

## 技术栈

- 纯 HTML + CSS + JavaScript
- [Chart.js 4.4](https://www.chartjs.org/)（通过 CDN 加载）
- 数据来源：`/api/stats`（统计）、`/api/visits`（访问记录）、`/api/device-labels`（设备标签）
- 无构建步骤

## 访问方式

- 导航页 ⚙ 设置 → 🔐 管理后台
- 直接访问 `/admin/`

## 操作说明

- **登录**: 输入密码（默认 `admin888`），后端验证 session
- **修改密码**: 点击🔑按钮，弹窗修改
- **设备打标**: 在访问记录表中点击设备标识（显示为截断的 device_id 或标签），弹窗输入标签名称
- **标签模式**: 点击「🏷 标签模式」按钮切换显示 device_id 或标签
