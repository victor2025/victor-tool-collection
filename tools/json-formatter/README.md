# JSON 工具

JSON 格式化、转义与反转义工具。纯浏览器端处理，不传输任何数据。

## 功能

### 格式化
- 纯文本 JSON 排版：缩进对齐、不改原始值、保留字符串原文
- **反转义嵌套 JSON**：勾选后，将 JSON 字符串字段中嵌套的 JSON 结构自动展开为真实结构体
- **层数控制**：可选择反转义的深度（1~20 层）
- 语法高亮输出

### 转义
- 将 JSON 对象/数组转换为 JSON 字符串格式
- 输出如 `"{\"key\":\"value\"}"` 的转义字符串

## 技术栈

- 原生 HTML / CSS / JavaScript
- 自实现 token 化 JSON 排版器
- 无外部依赖
