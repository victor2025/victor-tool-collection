# ⏰ 时间戳转换工具

Unix 时间戳与日期时间的双向转换工具，支持秒和毫秒两种单位。

## 功能

- **当前时间戳** — 实时显示当前 Unix 时间戳（秒/毫秒切换），同步显示 UTC 和北京时间
- **日期 → 时间戳** — 输入日期时间，自动转换为对应的 Unix 时间戳
- **时间戳 → 日期** — 输入时间戳，自动转换为可读的 UTC 和北京时间

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无外部依赖
- 无构建步骤
